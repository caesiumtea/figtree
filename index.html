Welcome to The Fig Tree!

This site is here as a helper to the Figment pet site, inspired by sites like Jellyneo, Animal Crossing Community, and the now-departed Tektek.org. Hopefully it can make your time playing Figment a little bit easier!

This site will go through a few iterations as I build up (and learn how to build!) new features. The first will be a listing of all shop items on a single page, so you can Ctrl+F to search them, with links to which shop page the item is on! Later on, The Fig Tree will become a site where you can save wishlists and search for users to trade with. At this stage, while Figment is still in alpha, the purpose of this site is basically to fill in for some of the functionality (like searching shop items) that Figment hasn't implemented quite yet in the main site.
